# Yifei Liu Personal Site

A lightweight static personal website inspired by the structure of Jon Higgins' portfolio:

- `index.html` - landing page with animated line cube.
- `work.html` - selected XR projects.
- `words.html` - paper reading and reproduction notes.
- `who.html` - short bio, interests, background, and contact links.

The site is plain HTML/CSS/JS and can be deployed directly with GitHub Pages.
