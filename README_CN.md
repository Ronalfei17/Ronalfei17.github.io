## 🚀 项目经历

### Liar VR：基于全身追踪的多人欺骗类 VR 卡牌游戏
**2026.02 - 2026.05** | 一个多人 VR 欺骗类卡牌游戏，通过全身追踪读取玩家的手势与肢体语言。

- [项目演示](https://www.youtube.com/watch?v=oPDAqGpKcfY&t=14s)
- **多人同步：** 基于 **Photon** 实现 VR 场景同步，并学习多人在线环境中的物体所有权切换机制。
- **全身交互：** 集成 **Unity Movement SDK**，使用 Meta Quest 3 的传感器数据驱动虚拟角色，实现手势交互与身体动作表达。
- **实体卡牌交互：** 使用 **Meta XR Interaction SDK** 搭建完整的物理卡牌抓取系统，增强欺骗类游戏的核心交互体验。

### AR Cocktail：沉浸式 AR 鸡尾酒点单系统
**2026.02 - 2026.05** | 一个沉浸式 AR 点单原型，用 3D 鸡尾酒模型呈现菜单内容，提升点单的可视化与空间体验。

- [项目短片](https://www.youtube.com/shorts/217OUgr1wzI)
- **AR 工作流：** 学习并应用 **AR Foundation** 基础能力，搭建增强现实点单交互流程。
- **3D 展示：** 探索在 AR 场景中展示鸡尾酒 3D 模型，让商品选择更直观、更具沉浸感。
- **空间体验设计：** 建立了对 AR 项目结构、场景配置与交互式展示设计的实践理解。

### Sonic Playground：VR 中的沉浸式空间音频体验
**2025.10 - 2025.12** | 在 Meta Quest 3 上开发交互式 VR 展览，用于探索声学物理现象。

- [代码](https://github.com/Ronalfei17/Yifei-Personal-Page/tree/Sonic-Playground) · [演示视频](https://www.bilibili.com/video/BV1KJFvz4ERD)
- **交互：** 使用 Meta XR Interaction SDK 实现精细化交互体验。
- **声学：** 集成 **Steam Audio** 实现实时声音传播效果。
